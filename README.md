# FocusFlow

FocusFlow is a minimalist study tracker focused on long-term consistency through automatic tracking and frictionless studying.

## Features

* Pomodoro or stopwatch timer modes
* Weighted daily activity suggestion engine
* Statistics dashboard
* Weekly goal and progress tracking
* CSV data export

## Technology

* Python
* NiceGUI
* SQLite
* Tailwind CSS
* Web Audio API

## How to Run and Test

You can run and test this application locally on your own computer by following these step-by-step steps:

### Prerequisites
Before starting, make sure you have **Python** installed on your system. If you do not have it, you can download and install it from the official website (python.org).

### Step 1: Open the Terminal or Command Prompt
* **Windows:** Press the `Windows Key`, type `cmd`, and press Enter.
* **Mac/Linux:** Open the `Terminal` application.
* Navigate to the folder where you downloaded this project's files (for example: `cd FocusFlow`).
