# FocusFlow

FocusFlow is a minimalist study tracker focused on long-term consistency through automatic tracking and frictionless studying.

## Features

* Pomodoro or stopwatch timer modes
* Weighted daily activity suggestion engine
* Statistics dashboard
* Weekly goal and progress tracking
* CSV data export

## Technology

* Python
* NiceGUI
* SQLite
* Tailwind CSS
* Web Audio API

## About

FocusFlow was developed as a minimalist personal project, aiming for a friction-free productivity system and tracking statistics to improve motivation.
